# GCxt-UNet: Efficient Deep Network for Medical Image Segmentation

#### GCtx-UNet is a U-shaped network architecture that incorporates the Global Context Vision Transformer (GC-ViT) to enhance medical image segmentation by effectively capturing both global and local features.
